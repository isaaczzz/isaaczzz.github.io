Camilla Nguyen Photography / camilla.photo

Portfolio website commissioned by independent photographer Camilla Nguyen. The site serves as a centralized digital portfolio on which Camilla is able to display photos and pricing and contact information. Web pages are served through a simple Node.js instance; all effects are pure HTML5 and CSS3 with the exception of the integrated nanoGALLERY jQuery plugin that pulls photos from targeted Flickr galleries. The result is a responsive site that the client is able to update easily with minimal interaction with the backend.

Visit camilla.photo.

client: Camilla Nguyen
medium: web
year: 2015
